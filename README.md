# qa-bug-report-web-ecommerce
Bug Report

Descrição:
Projeto de identificação e documentação de bugs em um sistema web de e-commerce.

Foram registrados defeitos com descrição detalhada, passos para reprodução, comportamento esperado, severidade e prioridade.

Este projeto simula o processo real de reporte de bugs no dia a dia de um QA, garantindo clareza na comunicação com o time de desenvolvimento.
